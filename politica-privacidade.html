Política de Privacidade

Esta página descreve, de forma simulada, como dados pessoais podem ser tratados dentro da plataforma Projeto Eu Abrigo.


1. Dados coletados
Podem ser coletados dados de cadastro, contato e informações fornecidas em formulários de adoção e agendamento.


2. Finalidade
Os dados são utilizados para organizar processos de adoção responsável, comunicação entre interessados e histórico básico de atendimentos.


3. Compartilhamento
Os dados podem ser compartilhados apenas com responsáveis do animal (abrigo ou cuidador) para viabilizar visitas e adoção.


4. Armazenamento e segurança
São adotadas medidas técnicas para reduzir riscos de acesso indevido. Ainda assim, nenhum sistema é totalmente isento de riscos.


5. Direitos do titular
O usuário pode solicitar atualização, correção ou exclusão de seus dados, conforme aplicável.
